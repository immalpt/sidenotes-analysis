## SideNotes 是什么来头？

SideNotes 是波兰开发者 Apptorium 出品的一款 Mac 侧边栏笔记工具。它的设计理念特别简单：把笔记栏贴在屏幕边缘，平时隐形，呼出时置顶，写完即走。
